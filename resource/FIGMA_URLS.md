# ParkingOn System - Figma 변환용 URL 목록

`html.to.design` 플러그인에서 사용할 URL 목록입니다.

## 🔗 사용 방법

1. Figma Desktop 열기
2. 파일 생성 또는 열기
3. Plugins → `html.to.design` 실행
4. 아래 URL을 복사하여 플러그인에 입력
5. "Import" 클릭

## ⚠️ 주의사항

1. **로그인 필요 페이지**
   - 로그인하지 않으면 로그인 페이지로 리다이렉트됩니다
   - 브라우저에서 먼저 로그인한 후, 같은 브라우저에서 플러그인 사용

2. **로컬 서버 실행**
   - 서버가 `http://localhost:8080`에서 실행 중이어야 합니다
   - 확인: 터미널에서 Spring Boot 실행 확인

3. **세션 유지**
   - 브라우저 세션이 유지된 상태에서 플러그인 사용
   - 필요시 각 URL을 브라우저에서 직접 열어 확인

4. **플러그인 제한사항**
   - JavaScript 동적 콘텐츠는 완벽하게 변환되지 않을 수 있음
   - 복잡한 인터랙션은 Figma에서 수동 조정 필요
   - 폰트가 다르게 렌더링될 수 있음

## 🎨 변환 후 작업

`html.to.design`으로 가져온 후:

1. **레이어 정리**
   - 불필요한 레이어 삭제
   - 레이어 이름 정리

2. **스타일 통일**
   - Color Styles 적용
   - Text Styles 적용
   - Effect Styles 적용

3. **컴포넌트화**
   - 반복되는 요소를 Component로 변환
   - Variants 설정 (버튼 상태, 테이블 행 등)

4. **Auto Layout 적용**
   - 유연한 레이아웃을 위해 Auto Layout 설정
   - Spacing과 Padding 조정

5. **프로토타이핑**
   - 페이지 간 연결
   - Interactive Component 설정

## 🔧 트러블슈팅

### 문제: "Cannot connect to localhost"
**해결책**:
- Spring Boot 서버 실행 확인
- 브라우저에서 URL 직접 접근 테스트

### 문제: "Unauthorized" 또는 로그인 페이지로 이동
**해결책**:
- 같은 브라우저에서 먼저 로그인
- 쿠키/세션 유지 확인

### 문제: 스타일이 깨짐
**해결책**:
- CSS 파일 로드 확인
- 브라우저 개발자 도구에서 에러 확인
- 필요시 수동으로 스타일 조정

### 문제: 폰트가 다름
**해결책**:
- Figma에서 시스템 폰트로 변경
- Text Styles 적용하여 통일
